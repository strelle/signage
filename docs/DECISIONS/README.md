# DECISIONS (ADR)

Architecture and operational decisions that affect maintainability, security, or delivery.

## How to Add

Create files as:

- `YYYY-MM-DD-<short-title>.md`

Use this template:

- Context
- Decision
- Consequences
- Alternatives considered
- Evidence/links

## Rule

If a change impacts deployment flow, security posture, ownership, or handoff behavior, add/update an ADR entry before merge.
